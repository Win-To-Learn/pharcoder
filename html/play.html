Welcome

Welcome Starcoder Captain, we need your help.

Your mission has 3 main parts:

  • Shoot Asteroids
  • Get the Seeds inside of the Asteroids
  • Plant Trees on the Planets

You will use code to gain more power.

First, get familiar with opening and closing the Coding window.

Click the "Open/Close Coding Window" button on the top of the screen 4 times, then scroll down and try the next challenge.

Changing the Turning Force

Let's change the turning force of your spaceship so you can control it better.


Change the turning force to 30.

  • Go to Blockly and select Starcoder.
  • Find the [set ship turning force to] block, then change the number to 30.

Remember to click SUBMIT CODE

See if your turning force is slower.

Turning on the Grid Mode

Let's change the view into grid mode so that you can navigate in a better way.

Press "G" to change your view to grid mode in order to switich to grid mode and see the coordinates.

Press "G" again to turn off the grid mode.
